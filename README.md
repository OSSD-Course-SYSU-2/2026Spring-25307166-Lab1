# 炼金工坊 - HarmonyOS游戏应用

## 项目简介
一款基于HarmonyOS的炼金术主题RPG游戏，玩家可以通过收集材料、调配配方、使用不同法阵来炼金，获得各种物品并完成委托任务。

## 核心功能

### 1. 主界面
- 显示玩家状态（金币、体力）
- 六大功能入口：委托、炼金、图鉴、冒险、公会、个人信息
- 自适应手机、平板、电脑屏幕

### 2. 炼金系统
**材料管理**
- 33种原材料，分为7种颜色类型（红、橙、黄、绿、青、蓝、紫）
- 每种材料都有独特的图案标识
- 初始材料数量：100份

**炼金流程**
1. 选择材料（最多100份）
2. 点击"开始炼金"消耗材料
3. 进入星盘转盘选择法阵
4. 根据法阵类型获得对应物品

**法阵系统**
- 8种法阵，每种都有独特图案和效果
- 赤焰：战斗灵药
- 翠风：社交香水
- 幽蓝：幸运护身符
- 紫月：大型体力药水
- 虚空混沌阵：远古祝福
- 生命神树阵：体力蛋糕
- 深渊召唤阵：凤凰之羽
- 星辰命运阵：材料磁石

### 3. 物品系统
**物品类型**
- 小型体力药水：恢复5点体力
- 大型体力药水：恢复15点体力
- 战斗灵药：战斗力+20
- 社交香水：交际+15
- 幸运护身符：幸运+25
- 材料磁石：掉落率+30%
- 体力蛋糕：恢复10点体力
- 远古祝福：全属性+10
- 凤凰之羽：失败时自动复活
- 金币袋：获得50-100金币

**物品获取**
- 通过炼金获得
- 根据法阵类型决定物品类型
- 物品等级由法阵复杂度决定

### 4. 委托系统
**委托流程**
1. 查看可接委托列表
2. 接受委托（需要指定材料）
3. 收集所需材料
4. 提交委托获得金币奖励

**委托特性**
- 委托由NPC发布
- 有截止时间限制
- 放弃委托会扣除20%金币作为惩罚
- 刷新可获得新委托

### 5. 冒险系统
**冒险地点**
- 多种冒险地点可选
- 不同地点有不同的难度和奖励

**冒险机制**
- 消耗体力进行冒险
- 可使用物品增强属性
- 根据骰子判定成功或失败
- 成功获得材料和金币奖励

### 6. 图鉴系统
**材料图鉴**
- 显示所有已解锁材料
- 点击查看详细信息（产地、属性、效果、传说、获取难度）
- 显示材料数量和独特图案

**物品图鉴**
- 显示所有炼金物品
- 点击查看详细信息（描述、稀有度、特性、使用场景）
- 显示拥有数量

### 7. 公会系统（NPC交互）
**NPC角色**
- 奥帕克：神秘的旅行商人
- 塔莉亚：年轻的女炼金师
- 其他炼金术师

**交互功能**
- 与NPC交流提升好感度
- 解锁NPC故事剧情
- 获得特殊配方奖励

## 技术特性

### 屏幕适配
- 自动识别设备类型（手机/平板/电脑）
- 动态调整字体大小、布局列数、图标尺寸
- 支持横竖屏切换

### 视觉效果
- Canvas绘制法阵图案
- 炼金釜动态效果（气泡、颜色混合）
- 材料独特图标标识
- 流畅的转盘动画

### 数据管理
- GameManager单例管理游戏状态
- 材料和物品数量实时更新
- 委托状态持久化

## 游戏流程

1. **新手起步**
   - 初始获得100份各种材料
   - 查看图鉴了解材料属性
   - 尝试第一次炼金

2. **炼金探索**
   - 选择材料组合（建议100份）
   - 选择合适法阵
   - 获得炼金物品

3. **委托任务**
   - 接受委托获取目标
   - 通过炼金或冒险收集材料
   - 提交委托赚取金币

4. **冒险挑战**
   - 使用物品增强属性
   - 挑战不同地点
   - 获得稀有材料

5. **NPC互动**
   - 与NPC交流提升好感
   - 解锁故事剧情
   - 获得特殊配方

## 注意事项

1. 炼金会消耗所有已选择的材料
2. 委托有截止时间，超时会失败
3. 冒险需要消耗体力，体力会自动恢复
4. 不同法阵适合不同的炼金目标
5. 与NPC互动可以获得稀有配方

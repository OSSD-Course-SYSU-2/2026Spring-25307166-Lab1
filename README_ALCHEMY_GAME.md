# 炼金术士集会 - HarmonyOS西幻魔法游戏

## 🎮 游戏简介

一个基于HarmonyOS框架开发的西幻风格炼金术士游戏，包含完整的冒险系统、炼金系统和NPC互动系统。

## ✨ 核心功能

### 1. 开场流程
- **黑屏开场** - 渐入文字介绍游戏背景
- **头像绘制** - 在屏幕上绘画生成角色头像
- **姓名输入** - 设定角色名称
- **身份确认** - 进入游戏世界

### 2. 六大功能区

#### 📜 委托系统
- 随机NPC发送委托任务
- 需求物品和奖励金币显示
- 接受或拒绝委托选择

#### 🔮 炼金系统
- **大锅显示** - 带气泡特效的炼金锅
- **七大色系材料** - 红、橙、黄、绿、青、蓝、紫
- **颜色混合** - 材料按比例实时混合
- **容量限制** - 总容量100，单材料上限100
- **法阵转盘** - 炼金后随机获得法阵

#### 📖 图鉴系统
- 书本样式展示
- 已解锁材料和物品查看
- 详细信息（产地、属性、效果、传说）

#### ⚔️ 冒险系统 (新增)
**四大冒险地点**：
- **天街** - 漂浮云端的神秘街道（红色系材料）
  - 危险等级：中等
  - 成本：50金币
  - 特色：高空飞行，云雾缭绕
  
- **地牢** - 深埋地下的古代监狱（蓝色系材料）
  - 危险等级：较高
  - 成本：100金币
  - 特色：阴暗潮湿，充满陷阱
  
- **悬崖** - 云雾缭绕的绝壁（绿色系材料）
  - 危险等级：高
  - 成本：150金币
  - 特色：陡峭险峻，风声呼啸
  
- **龙穴** - 远古巨龙的巢穴（紫色系材料）
  - 危险等级：极高
  - 成本：300金币
  - 特色：龙焰残留，宝光闪烁

**遭遇事件**：
- 神秘商人 - 出售稀有材料
- 流浪炼金师 - 分享炼金秘闻
- 危险生物 - 战斗或逃脱
- 隐藏宝箱 - 意外收获

#### 🏛️ 公会系统 (新增)
**三位固定NPC**：

**奥帕克** 🎭
- 神秘的旅行商人，总是戴着面具
- 性格：谨慎、神秘
- 任务：要求稀有物品，奖励丰厚
- 好感故事：从初次相遇到永恒羁绊

**塔莉亚** 👩‍🔬
- 年轻的女炼金师，充满好奇心
- 性格：热情、好学
- 任务：要求实验性物品，奖励中等
- 好感故事：从好奇到传承

**帕帕拉查** 🧙‍♂️
- 年长的炼金大师，严厉但公正
- 性格：严厉、公正
- 任务：要求高品质物品，奖励极高
- 好感故事：从严厉到传承

**好感度系统**：
- 完成委托增加好感度
- 每级好感解锁专属故事
- 好感度影响奖励倍率

#### 👤 用户功能
- 角色信息展示
- 材料背包（按色系分类）
- 物品背包
- 金币数量
- 注销和重启

### 3. 四大法阵系统

#### 🔥 赤焰法阵
- 图案：火焰六芒星
- 对应：红橙黄系材料
- 效果：50%概率生成神级物品
- 描述：火焰般的六芒星，蕴含炽热之力

#### 🌿 翠风法阵
- 图案：层层涟漪
- 对应：绿青系材料
- 效果：50%概率生成神级物品
- 描述：层层涟漪，自然的呼吸

#### 💎 幽蓝法阵
- 图案：深邃漩涡
- 对应：蓝青系材料
- 效果：50%概率生成神级物品
- 描述：深邃的漩涡，连接深渊

#### 🌙 紫月法阵
- 图案：神秘封印
- 对应：紫蓝系材料
- 效果：50%概率生成神级物品
- 描述：神秘的封印，月光凝结

### 4. 物品配方系统

**常见物品**：
- 🧪 生命药剂 - 红70% + 橙30%
- 💎 魔力石 - 黄70% + 绿30%
- 🧫 解毒瓶 - 青40% + 蓝40% + 紫20%
- 💧 无魔水 - 任意比例（失败产物）

**稀有物品**：
- 💠 寻物宝石 - 蓝60% + (紫或青)40%
- ☠️ 诅咒粉尘 - 黄30% + 绿40% + (紫或蓝)30%
- 🌾 丰收瓶 - (红或橙)60% + 绿40%
- 💡 短时灵感瓶 - 绿50% + (青或蓝)50%

**传说物品**：
- 🔮 梦珍珠 - 红10% + 青50% + 紫40%
- 💍 恋戒 - 红30% + 橙40% + 紫30%
- 🍯 回忆药水 - 绿40% + 青30% + 蓝30%

## 🚀 运行方式

1. 在DevStudio中打开项目
2. 确保HarmonyOS SDK已配置
3. 运行到模拟器或真机
4. 游戏自动从开场动画开始

## 📁 项目结构

```
entry/src/main/ets/
├── model/                    # 数据模型
│   ├── GameTypes.ets         # 游戏类型定义
│   ├── Material.ets          # 22种材料数据
│   ├── Formation.ets         # 4种法阵定义
│   ├── AlchemyRecipe.ets     # 物品配方系统
│   ├── Adventure.ets         # 冒险系统
│   ├── NPC.ets               # NPC和好感系统
│   ├── GameState.ets         # 游戏状态
│   └── GameManager.ets       # 游戏管理器
└── pages/                    # 页面组件
    ├── GameEntry.ets         # 游戏入口
    ├── IntroPage.ets         # 开场动画
    ├── DrawingPage.ets       # 头像绘制
    ├── NameInputPage.ets     # 姓名输入
    ├── IdentityConfirmPage.ets # 身份确认
    ├── MainPage.ets          # 主页
    ├── CommissionPage.ets    # 委托页面
    ├── AlchemyPage.ets       # 炼金页面
    ├── FormationWheelPage.ets # 法阵转盘
    ├── CollectionPage.ets    # 图鉴页面
    ├── AdventurePage.ets     # 冒险页面 (新增)
    ├── NPCPage.ets           # NPC页面 (新增)
    └── UserPage.ets          # 用户页面
```
